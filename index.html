Lecturing

  • 2021-2022 Semester 2: MATH3817 Dynamic Programming and Stochastic Programming

    This course provides an introduction to the basic principles, classical models, and various applications of dynamic programming and stochastic programming

  • 2021-2022 Semester 1: MATH1025 Introduction to Mathematics and Statistics

    This course introduces basic mathematics and statistics for university students. The mathematics part covers elementary calculus, optimization, numerical methods and applications in finance, while the statistics part explains basic statistical notions, including confidence intervals, tests of hypotheses and linear regression, and discusses misuse and abuse of statistics.
